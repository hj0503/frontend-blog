1、sass简介

Sass是一款强化CSS的辅助工具，它在CSS语法的基础上增加了变量variables、嵌套nested rules、混合mixins、导入inline imports 等高级功能，这些拓展令CSS更加强大与优雅。使用Sass以及Sass的样式库（如 Compass）有助于更好地组织管理样式文件，以及更高效地开发项目。

  • this.app 表⽰根 Vue 实例
  • this.apps 保存所有⼦组件的 Vue 实例
  • this.options 保存传⼊的路由配置
  • this.beforeHooks 、 this.resolveHooks 、 this.afterHooks 表⽰⼀些钩⼦函数
